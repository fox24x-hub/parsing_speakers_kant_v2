# Skill: find-speakers-summer

## Purpose
Уточняет работу find-speakers-core для летнего сезона: бег, трейл, триатлон и другие летние активности.

## When to use
Используй вместе с find-speakers-core, когда:
- season = "лето"
- или пользователь явно упоминает бег, трейл, триатлон, велосипед, летние старты.

## Summer focus
Отдавай приоритет спикерам по темам:
- шоссейный бег (5 км, 10 км, полумарафон, марафон);
- трейлраннинг, горный бег;
- триатлон, дуатлон;
- велосипедные тренировки для бегунов и триатлетов;
- ОФП и функциональный тренинг для бегунов;
- подготовка к летним стартам и сериям забегов.

## Instructions
1. Сохраняй формат JSON из find-speakers-core.
2. Не добавляй тренеров, специализирующихся только на зимних видах спорта, если нет летнего контекста.
3. В `expertise` указывай, для каких дистанций/типов стартов спикер особенно полезен.
4. Если регион небольшой, допускается расширение географии согласно правилу geo-urals (от города к региону, УрФО, России).

## Output format
Такой же JSON, как у find-speakers-core, с явным акцентом на летний спорт в полях `sport` и `expertise`.
