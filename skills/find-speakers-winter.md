# Skill: find-speakers-winter

## Purpose
Уточняет работу find-speakers-core для зимнего сезона: приоритет зимних видов спорта и зимних мероприятий.

## When to use
Используй вместе с find-speakers-core, когда:
- season = "зима"
- или пользователь явно упоминает зимние виды спорта, снег, лыжи, сноуборд.

## Winter focus
Отдавай приоритет спикерам по темам:
- беговые лыжи;
- горные лыжи;
- сноуборд;
- ски-тур, фрирайд;
- зимний бег, подготовка к зимнему сезону;
- тренировки на снегу и льду.

## Instructions
1. Сохраняй общий формат JSON из find-speakers-core.
2. Не включай чисто «летних» экспертов (например, только по шоссе-бегу без зимнего контекста), если нет прямой связи с зимой.
3. Учитывай города и регионы с развитой зимней инфраструктурой (Екатеринбург, Челябинск, Пермь, Тюмень и др., если радиус поиска это позволяет).
4. В `expertise` чётко указывай, почему спикер полезен именно для зимних лекций KANT (подготовка к сезону, техника, экипировка, безопасность).

## Output format
Тот же JSON, что у find-speakers-core.

Убедись, что в `sport` и `expertise` явно виден зимний контекст.
