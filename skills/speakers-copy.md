# Skill: speakers-copy

## Purpose
Преобразует JSON со списком спикеров в читаемое сообщение для Telegram-бота parsing_speakers_kant_v2.

## When to use
Используй этот навык после того, как:
- find-speakers-core (и связанные навыки) вернули валидный JSON с массивом `speakers`;
- массив `speakers` не пустой.

## Input
JSON вида:

{
  "season": "...",
  "location_scope": "...",
  "speakers": [ ... ]
}

## Style guidelines
1. Короткий дружелюбный заголовок:
   - Пример: `Нашёл спикеров для зимних лекций в Екатеринбурге:`
2. Для каждого спикера используй компактный блок:
   - первая строка: `Имя Фамилия — вид спорта`
   - вторая строка: краткая экспертиза (1–2 предложения)
   - третья строка (опционально): `Профиль: <url>`, если url не пустой
3. Между спикерами оставляй пустую строку для читаемости.
4. Избегай длинных абзацев, пиши просто и по делу.

## Output format
Готовый текст для отправки в Telegram одним сообщением.

Пример:

Нашёл спикеров для зимних лекций в Екатеринбурге:

Иван Петров — беговые лыжи  
Тренер по беговым лыжам, готовит любителей к стартам и помогает с выбором экипировки KANT.  
Профиль: https://...

Анна Смирнова — зимний бег  
Специалист по подготовке к зимним забегам, рассказывает о тренировках на снегу и безопасности.  
Профиль: https://...

Не добавляй технических комментариев, только текст для пользователя.
